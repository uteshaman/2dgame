# 2D Console Game на Go

## Описание
Это простая **2D консольная игра**, написанная на Go с использованием библиотеки [termbox-go](https://github.com/nsf/termbox-go).  

Игрок управляет персонажем на игровом поле, собирает еду и набирает очки.  
Проект демонстрирует базовые навыки backend-разработки и работы с Go:

- структуры и методы (struct, methods)
- работа с каналами (goroutines, channels)
- игровой цикл (game loop)
- обработка пользовательского ввода
- разделение логики (main / game / render)

---

## Геймплей
- Игрок передвигается по полю  
- На карте появляется еда (красная клетка)  
- При сборе еды увеличивается **Score**  
- Еда появляется в случайном месте  

---

## Управление
- ↑ Стрелка вверх – движение вверх  
- ↓ Стрелка вниз – движение вниз  
- ← Стрелка влево – движение влево  
- → Стрелка вправо – движение вправо  
- Q / ESC / Ctrl+C – выход из игры
